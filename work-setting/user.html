사용자 관리 ​

시스템에 접속하는 구성원 계정을 생성하고, 각 사용자의 활동 현황을 통합적으로 관리하는 사용자 페이지에 대한 가이드입니다.

1. 화면 개요 ​

사용자 관리 화면은 전사적인 사용자 목록을 제공하며, 특정 사용자가 어떤 프로젝트에 참여하고 있는지, 할당된 업무의 처리 현황은 어떠한지 등 상세한 프로필 정보를 확인하고 관리할 수 있는 기능을 제공합니다.

2. 주요 기능 안내 ​

👥 사용자 목록 및 검색 ​

  • 전체 목록 조회: 시스템에 등록된 모든 사용자의 이름, 아이디, 이메일, 가입일 등을 한눈에 확인합니다.
  • 사용자 식별: 목록에서 특정 사용자를 클릭하여 해당 사용자의 상세 활동 대시보드로 이동할 수 있습니다.

🔍 사용자 상세 프로필 (User View) ​

특정 사용자를 선택하면 해당 인원과 관련된 모든 업무 데이터를 집계하여 보여줍니다.

  • 참여 프로젝트: 해당 사용자가 구성원으로 포함되어 있는 모든 프로젝트 리스트를 확인합니다.
  • 업무 통계: 사용자에게 할당된 업무의 총 개수와 완료/진행 중인 업무의 비중을 시각적으로 파악합니다.
  • 최근 활동 기록: 해당 사용자가 최근에 생성하거나 수정한 업무, 게시글 등 최신 활동 로그(최대 10건)를 추적합니다.

📝 계정 생성 및 정보 수정 ​

  • 사용자 생성 (): 새로운 구성원의 계정을 생성하고 기본 인적 사항을 등록합니다.
  • 정보 업데이트: 사용자의 이메일, 연락처, 권한 상태 등 프로필 정보를 최신으로 유지합니다.

3. 관리자 이용 팁 ​

  • 업무 부하 파악: 사용자 상세 보기의 업무 통계를 통해 특정 팀원에게 업무가 과도하게 집중되어 있지는 않은지 확인하여 업무를 재배분할 수 있습니다.
  • 활동 추적: 활동 로그를 활용하면 특정 이슈의 변경 이력이나 사용자의 최근 작업 흐름을 파악하는 데 도움이 됩니다.
  • 권한 관리: 사용자가 퇴사하거나 부서가 변경된 경우, 사용자 수정 기능을 통해 계정의 활성화 상태나 접근 권한을 즉시 조정하십시오.
