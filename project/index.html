프로젝트 등록 ​

프로젝트 등록 페이지는 선택한 프로젝트의 기본 정보를 조회하고, 프로젝트를 신규 등록하거나 수정할 수 있는 화면입니다.

  • 위치: 프로젝트 설정 > 프로젝트 등록

주요 기능 ​

  • 프로젝트 기본 정보 조회
  • 신규 프로젝트 등록
  • 프로젝트 정보 수정

화면 구성 ​

1. 프로젝트 선택 ​

프로젝트를 선택할 수 있습니다.

2. 프로젝트 개요 ​

선택한 프로젝트의 주요 정보를 표시합니다.

3. 작업 버튼 ​

  • 수정하기: 선택한 프로젝트의 정보를 수정할 수 있는 입력 폼으로 전환합니다.
  • 등록하기: 새로운 프로젝트를 등록하기 위한 빈 입력 폼으로 전환합니다.

4. 프로젝트 입력 폼 ​

프로젝트 등록 및 수정 시 사용하는 입력 영역입니다. 특히 사업 일정 섹션은 시스템의 자금 집행 현황(캐시플로우) 산정에 직접적인 영향을 미치므로 정확한 입력이 필요합니다.

주요 입력 항목 ​

라벨필수입력 여부비고
업무 프로젝트필수업무시스템 연계를 위한 프로젝트 선택. (+) 버튼을 클릭하여 새 업무 프로젝트를 즉시 생성 가능
프로젝트명필수프로젝트 명칭 입력 (최대 30자)
정렬순서프로젝트 목록 표시 순서 (미입력 시 등록 순서 기준)
프로젝트종류필수아파트, 타운하우스, 주상복합, 오피스텔, 근생, 숙박시설, 지식산업센터 등 선택
사업개시년도필수프로젝트의 공식 사업 개시 연도
직영운영여부본사 직접 운영(시행/PM) 여부 설정
동호지정여부분양/모집 시 동호수 지정 관리 여부
토지환지여부환지방식 도시개발사업구역 포함 여부
월별집계시작일필수캐시플로우 집계 시작월. 지정일 이전 데이터는 누계로 합산 표시됨
착공(예상)일필수공사기간의 시작일 (캐시플로우 월별 집계 기준일)
공사기간필수공사 소요 개월 수 입력
대지위치사업부지 대표 지번 등 위치 정보
용도지역지구필수도시계획상 용도지역/지구 정보
건축규모필수지하 층수 ~ 지상 층수 등 건축 규모
세대(호/실)수총 공급 세대수 또는 호/실수
면적 관련 항목제곱미터 단위 입력, 소수점 4자리까지 정밀도 지원 (대지, 연면적, 건축면적 등)
용적율/건폐율% 단위 입력, 소수점 4자리까지 지원
법정/계획주차법정 및 실제 계획된 주차 대수
