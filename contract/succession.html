권리 의무 승계 ​

계약의 권리 및 의무를 양도/양수하는 승계 정보를 관리하는 페이지입니다.

  • 위치: 계약 정보 관리 > 권리 의무 승계

개요 ​

특정 계약에 대한 권리 의무 승계 내역을 조회하고, 새로운 승계 건을 등록하거나 기존 승계 정보를 수정할 수 있습니다. 양도 계약자 및 양수 계약자의 정보를 입력하고, 승계 관련 일자 및 변경 인가 여부를 관리합니다.

주요 기능 ​

  • 계약자 검색 및 선택 기능 (양도 계약자 지정)
  • 승계 진행 건 목록 조회 및 엑셀 출력
  • 새로운 권리 의무 승계 등록
  • 기존 권리 의무 승계 정보 수정 및 삭제
  • 양도 계약자 및 양수 계약자 정보 관리
  • 승계 신청일, 매매 계약일, 변경 인가일 등 주요 일자 관리
  • 변경 인가 여부 설정
  • URL 파라미터를 통한 특정 승계 건 하이라이트 및 페이지 이동 지원

화면 구성 ​

1. 계약자 검색 및 선택 ​

양도 계약자를 검색하고 선택할 수 있는 기능을 제공합니다. 검색 결과는 버튼 형태로 표시되어 쉽게 선택할 수 있습니다.

주요 기능 ​

  • 검색 입력 필드: 계약자명, 비고, 계약 일련번호로 검색
  • 검색 버튼: 검색 실행
  • 검색 결과 목록: 검색된 계약자 목록을 버튼 형태로 표시, 클릭 시 해당 계약자를 양도 계약자로 설정하고 상세 정보 로드

2. 계약 관리 내비게이션 ​

계약 상세 관리, 권리 의무 승계, 계약 해지 관리 페이지로 이동할 수 있는 탭 형태의 내비게이션을 제공합니다. 현재 활성화된 페이지를 시각적으로 표시합니다.

주요 항목 ​

  • 계약 상세 관리 버튼
  • 권리 의무 승계 버튼
  • 계약 해지 관리 버튼

3. 계약자 정보 알림 ​

현재 선택된 양도 계약자의 주요 정보(계약자명, 현재 상태)를 알림 형태로 표시합니다. 계약자 선택 해제 기능과 함께, 승계 진행 중이거나 해지된 계약자에 대한 특별 알림을 제공합니다.

주요 표시 항목 ​

  • 계약자명
  • 현재 상태 (청약, 계약, 해지신청, 해지완료 등)
  • 승계 진행 중 여부
  • 계약자 선택 해제 버튼

4. 승계 등록/수정 버튼 ​

새로운 승계 건을 등록하거나, 현재 조회 중인 승계 건을 수정하기 위한 버튼을 제공합니다. 승계 진행 상태에 따라 버튼의 텍스트와 색상이 변경됩니다.

주요 기능 ​

  • '등록하기' 또는 '수정하기' 버튼 클릭 시 권리 의무 승계 등록/수정 폼 모달 호출.

5. 승계 진행 건 목록 ​

등록된 권리 의무 승계 건들의 목록을 테이블 형태로 표시합니다. 각 승계 건에 대한 상세 정보와 확인 버튼을 제공합니다.

주요 표시 항목 ​

항목설명
계약 정보승계 대상 계약의 일련번호
양도계약자권리를 양도하는 계약자의 이름
양수계약자권리를 양수받는 계약자의 이름
승계신청일승계 신청이 접수된 날짜
매매계약일양도/양수 계약이 체결된 날짜
변경인가일승계에 대한 변경 인가가 완료된 날짜
변경인가여부승계에 대한 변경 인가 완료 여부
확인승계 상세 정보를 확인하거나 수정하는 버튼

6. 권리 의무 승계 등록/수정 폼 ​

새로운 권리 의무 승계 정보를 등록하거나 기존 정보를 수정하는 모달 폼입니다. 양도 계약자 정보는 자동으로 채워지며, 양수 계약자의 상세 정보를 입력합니다.

주요 입력 항목 ​

  • 양도계약자: (자동 입력) 권리를 양도하는 계약자 정보.
  • 승계신청일: 필수. 승계 신청이 접수된 날짜.
  • 매매계약일: 필수. 양도/양수 계약이 체결된 날짜.
  • 양수계약자 인적사항:
    • 이름: 필수. 양수 계약자의 성명.
    • 생년월일: 필수. 양수 계약자의 생년월일.
    • 성별: 필수. 양수 계약자의 성별.
    • 휴대전화: 필수. 양수 계약자의 휴대전화 번호.
    • 집전화, 기타연락처, 이메일: 양수 계약자의 추가 연락처 정보.
    • 주민등록주소: 필수. 양수 계약자의 주민등록 주소 (우편번호 API 연동).
    • 우편송달주소: 필수. 양수 계약자의 우편물 수령 주소 (주민등록 주소와 동일 설정 가능, 우편번호 API 연동).
  • 변경인가여부: 변경 인가 완료 여부 체크박스.
  • 변경인가일: 변경 인가 완료 시 날짜 입력.
  • 비고: 기타 특이사항.
