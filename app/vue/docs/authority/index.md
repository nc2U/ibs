# 회사 정보 관리

시스템의 운영 주체가 되는 회사의 기본 정보(사업자 정보, 법인 정보, 주소 등)를 등록하고 관리하는 페이지입니다.

- **위치**: `환경 설정 > 회사 정보 관리`

## 개요

회사의 법적, 행정적 기초 정보를 관리합니다. 여기서 등록된 정보는 각종 문서의 발신인 정보, 세무 신고 자료, 그리고 시스템 내의 각종 서식에서 회사를 식별하는 기준 데이터로 사용되므로 정확한 관리가 필요합니다.

## 주요 기능

- **회사 기본 정보 조회**: 회사명, 대표자명, 사업자 및 법인 등록번호 등 핵심 정보를 한눈에 확인
- **신규 회사 등록**: 시스템 도입 초기 또는 운영 주체 변경 시 새로운 회사 정보 생성 (최고 관리자 권한 필요)
- **정보 업데이트**: 주소 이전, 대표자 변경 등 회사 정보의 변동 사항 발생 시 즉시 업데이트 지원
- **입력 유효성 검사**: 사업자등록번호 및 법인번호의 정해진 형식을 강제하여 데이터 오류 방지
- **주소 검색 연동**: 다음(Daum) 우편번호 서비스와 연동하여 정확한 우편번호와 주소 입력 지원
- **권한 기반 제어**: 회사 정보의 중요성을 고려하여 쓰기 권한(CompanySettingsAuthGuard)이 있는 사용자만 수정 가능

## 화면 구성

### 1. 회사 정보 상세 (Detail)
현재 시스템에 등록된 회사의 상세 데이터를 표 형식으로 표시합니다.

- **표시 항목**: 회사명, 대표자명, 사업자등록번호, 법인등록번호, 업태, 종목, 설립일자, 개업일자, 회사 주소
- **관리 버튼**: [수정하기] 버튼을 통해 편집 화면으로 이동하며, 정보가 없는 경우 [등록하기] 버튼이 활성화됩니다.

### 2. 회사 정보 양식 (Form)
회사 정보를 새로 입력하거나 수정하는 화면입니다.

- **기본 정보 입력**: 회사명부터 개업일자까지 모든 필드를 입력하며, 필수 항목은 별도로 표시됩니다.
- **번호 마스킹**: 사업자등록번호(###-##-#####)와 법인등록번호(######-#######)는 입력 시 자동으로 형식이 교정됩니다.
- **주소 입력**: [우편번호] 버튼을 클릭하여 주소를 검색하면 우편번호와 기본 주소가 자동 입력되며, 상세 주소만 추가로 작성합니다.

## 💡 주요 관리 포인트

### 데이터의 정확성
사업자등록번호와 법인등록번호는 세무 및 법적 서류에 그대로 인용되는 경우가 많습니다. 오타가 발생할 경우 연동된 다른 기능(예: 세금계산서 발행 등)에서 문제가 생길 수 있으므로 사업자등록증과 대조하여 정확히 입력하십시오.

### 최고 관리자 권한 (SuperAuth)
회사 정보의 **신규 등록** 및 **삭제**는 시스템 보안과 데이터 무결성을 위해 최고 관리자(SuperUser) 권한을 가진 사용자에게만 허용됩니다. 일반 관리자는 허용된 범위 내에서 '수정'만 가능합니다.

### 주소 정보의 최신화
회사 주소는 각종 공문서의 기본 주소로 사용됩니다. 사무실 이전 시 주소 정보를 즉시 업데이트하여 출력물이나 문서상에 구 주소가 표기되지 않도록 주의하십시오.

### 정산 및 설립일자 관리
설립일자와 개업일자는 회사의 연혁 관리뿐만 아니라 시스템 내의 각종 통계 및 정산 시작 시점의 참고 자료가 될 수 있으므로 정확한 날짜를 지정하십시오.
