# 예산 정보 설정

프로젝트의 전체 수입 및 지출 예산을 등록하고 관리합니다. 등록된 예산 정보는 향후 자금 집행 및 수납 현황 분석의 기준 데이터로 활용됩니다.

## 수입 예산 등록

분양 수입이나 기타 유입 자금에 대한 예산을 계정과목, 차수, 타입별로 세분화하여 등록합니다.

- **위치**: `프로젝트 설정 > 예산 정보 설정 > 수입 예산 등록`

#### 주요 입력 항목

| 라벨 | 필수입력 여부 | 비고 |
| :--- | :---: | :--- |
| 계정과목 | 필수 | 분양수입, 연체료수입 등 해당 수입의 계정 선택 |
| 차수 | | 수입 예산을 특정 계약 차수와 연결할 경우 선택 |
| 타입 | | 수입 예산을 특정 유닛 타입(평형)과 연결할 경우 선택 |
| 항목명칭 | | 타입 미선택 시 필수 입력. 해당 수입 항목의 구체적 명칭 |
| 단위 세대 단가 | | 해당 타입 또는 항목의 1개 단위당 평균 가격 |
| 수량 | 필수 | 공급 세대수 또는 해당 수입 항목의 총 수량 |
| 기초 수입 예산 | 필수 | 최초 사업 승인(인준) 시의 수입 예산 금액 |
| 현황 수입 예산 | | 사업 진행 중 변경된 현재 시점의 수입 예산 금액 |

## 지출 예산 등록

프로젝트 수행을 위해 발생하는 각종 공사비, 사업비 등 지출 항목에 대한 예산을 등록합니다.

- **위치**: `프로젝트 설정 > 예산 정보 설정 > 지출 예산 등록`

#### 주요 입력 항목

| 라벨 | 필수입력 여부 | 비고 |
| :--- | :---: | :--- |
| 계정 과목 | 필수 | 공사비, 설계비, 마케팅비 등 지출 계정 선택 |
| 산출 근거 | | 해당 예산 금액이 산출된 구체적인 기준이나 내역 요약 |
| 기초 지출 예산 | 필수 | 최초 사업 승인(인준) 시의 지출 예산 금액 |
| 현황 지출 예산 | | 사업 진행 중 변경된 현재 시점의 지출 예산 금액 |

> **참고**: 수입/지출 예산 모두 '기초 예산'은 초기 사업 계획상의 수치를, '현황 예산'은 물가 상승이나 설계 변경 등으로 인해 조정된 실제 관리 수치를 입력하여 예산 대비 실행 현황을 비교 관리할 수 있습니다.
