# 분양 조건 설정

프로젝트의 납부 일정, 계약금 및 공급 가격, 구비 서류, 유상 옵션 등 분양 계약에 직접적으로 적용되는 상세 조건을 설정합니다.

## 납부 회차 등록

계약금, 중도금, 잔금 등 분양 대금의 납부 회차를 정의하고, 각 회차별 납부 기한 및 선납할인/연체가산 기준을 설정합니다.

- **위치**: `프로젝트 설정 > 분양 조건 설정 > 납부 회차 등록`

#### 주요 입력 항목

| 라벨         | 필수입력 여부 | 비고                                      |
|:-----------|:-------:|:----------------------------------------|
| 타입 종류      |   필수    | 공동주택, 오피스텔 등 시설 용도별 회차 구분               |
| 회차 종류      |   필수    | 계약금, 중도금, 잔금, 업무대행비 등 항목 선택             |
| 계약금 산정방법   |         | (계약금인 경우) 등록 계약 금액 우선 또는 분양가 대비 비율 중 선택 |
| 납입회차 코드    |   필수    | 동일 회차 식별 코드 (중복 등록을 통한 분할 납부 가능)        |
| 납부순서       |   필수    | 동일 회차 내에서의 수납 순서                        |
| 회차 명 / 별칭  |   필수    | 고지서 등에 표시될 회차 명칭                        |
| 납부 약정금액    |         | 정액으로 정해진 경우 입력 (예: 업무대행비)               |
| 납부비율(%)    |         | 분양가 대비 납부 비율 (잔금은 자동 계산)                |
| 납부 약정일     |         | 고정된 납부 기한이 있는 경우 입력                     |
| 전회 기준 경과일수 |         | 전 회차(예: 계약일)로부터 일정 기간 후 납부 시 설정         |
| 선납/연체 설정   |         | 선납할인율, 연체가산율 및 각각의 적용 기준일 설정            |

## 계약 금액 등록

차수 및 타입별로 고정된 계약 금액을 설정합니다.

- **등록 대상**: 동호 미지정 계약이나 타입별(층별 구분 없이) 일괄 정액제 계약금을 운영할 때 등록합니다.
- **적용 우선순위**: 계약금 회차의 금액은 다음 순서로 결정됩니다.
    1. **회차별 고정 금액**: [납부 회차 등록]에서 직접 입력한 '납부 약정금액'
    2. **특별 약정 금액**: [공급 가격 등록]의 '특별약정 추가'로 등록된 금액
    3. **타입별 계약 금액**: 본 메뉴([계약 금액 등록])에서 설정한 정액 금액 (납부비율보다 우선 적용)
    4. **납부 비율 계산**: [납부 회차 등록]의 '납부비율(%)'을 공급가에 곱한 금액
- **참고**: 각 유닛별 공급 가격의 일정 비율을 계약금으로 정하는 일반적인 경우에는 [납부 회차 등록]의 납입 비율이 자동 적용되므로 별도로 이 데이터를 등록할 필요가 없습니다.

- **위치**: `프로젝트 설정 > 분양 조건 설정 > 계약 금액 등록`

#### 주요 입력 항목

| 라벨       | 필수입력 여부 | 비고                    |
|:---------|:-------:|:----------------------|
| 차수       |   필수    | 대상 계약 차수 그룹 선택        |
| 타입       |   필수    | 대상 유닛 타입 선택           |
| 회차별 납부금액 |   필수    | 해당 차수/타입에 적용할 정액 계약금액 |

## 공급 가격 등록

동/타입/층별 범위에 따른 실제 공급 가격을 등록합니다. 건물가, 대지가, 부가세를 구분하여 관리합니다.

- **위치**: `프로젝트 설정 > 분양 조건 설정 > 공급 가격 등록`
- **산출 우선순위**: 시스템은 다음 순서에 따라 계약 가격을 결정합니다.
    1. **개별 계약 가격**: 특정 계약건에 별도로 입력된 가격 (최우선)
    2. **기준 공급 가격**: 본 메뉴([공급 가격 등록])에서 설정한 차수/타입/층별 가격
    3. **프로젝트 예산 단가**: [수입 예산 등록]에서 설정한 항목별 평균 단가
    4. **타입별 평균 단가**: [타입 정보 등록]에서 설정한 평균 가격

#### 주요 입력 항목

| 라벨           | 필수입력 여부 | 비고                                |
|:-------------|:-------:|:----------------------------------|
| 구분 / 차수 / 타입 |   필수    | 가격을 등록할 상세 범위 선택                  |
| 층별 조건        |   필수    | 등록된 층 범위(저층, 기준층 등) 선택            |
| 건물가 / 대지가    |   필수    | 각 항목별 금액 입력 (단위: 원)               |
| 부가세          |   필수    | 건물분에 대한 부가세 입력                    |
| 기준공급가        |   필수    | 건물가 + 대지가 + 부가세 합계 (자동 계산)        |
| 특별약정 추가      |         | 특정 회차에 대해 비율이 아닌 확정 금액을 적용할 경우 설정 |

#### 전체 계약건 공급가 재설정 (관리자 전용)

공급 가격이나 계약 금액 정보를 수정한 후, 이미 등록된 모든 계약 데이터에 변경된 내용을 일괄 반영할 때 사용합니다.

- **미리보기**: 실제 반영 전 업데이트 대상 계약 건수와 샘플 데이터를 콘솔 등을 통해 확인합니다.
- **전체 공급가 재설정**: 현재 등록된 최신 공급 가격 및 계약 금액 데이터를 기반으로 모든 계약 건의 가격 정보를 강제로 재설정합니다.
- **주의사항**: 이 작업은 복구가 불가능하며 등록된 모든 계약 데이터의 수납 약정 금액이 변경될 수 있으므로 매우 신중하게 실행해야 합니다.

## 구비 서류 등록

계약 시 계약자가 제출해야 하는 증빙 서류 및 동의서 목록을 관리합니다.

- **위치**: `프로젝트 설정 > 분양 조건 설정 > 구비 서류 등록`

#### 주요 입력 항목

| 라벨         | 필수입력 여부 | 비고                         |
|:-----------|:-------:|:---------------------------|
| 서류구분       |   필수    | 증빙서류 또는 동의서류(약정서 등) 중 선택   |
| 서류 유형      |   필수    | 주민등록등본, 인감증명서 등 세부 유형 선택   |
| 필요 수량      |         | 제출 받아야 할 서류 부수             |
| 필수 여부      |   필수    | 필수, 선택, 조건부 필수 중 선택        |
| 설명 / 표시 순서 |         | 서류 지참 안내 시 참고사항 및 목록 노출 순서 |

## 옵션 품목 등록

발코니 확장, 시스템 에어컨, 가전제품 등 수분양자가 선택할 수 있는 유상 옵션 품목을 관리합니다.

- **위치**: `프로젝트 설정 > 분양 조건 설정 > 옵션 품목 등록`

#### 주요 입력 항목

| 라벨         | 필수입력 여부 | 비고                               |
|:-----------|:-------:|:---------------------------------|
| 타입구분       |   필수    | 해당 옵션을 선택할 수 있는 대상 타입 (다중 선택 가능) |
| 품목 코드 / 이름 |   필수    | 옵션 품목 식별 정보                      |
| 세대 옵션      |         | 규격, 색상 등 세부 선택 사항                |
| 제조사        |         | 품목의 제조사 또는 브랜드                   |
| 옵션 가격      |   필수    | 총 판매 가격                          |
| 계약금 / 잔금   |         | 옵션 대금의 분납 시 각 금액 설정              |

## 💡 산출 로직 요약 안내

IBS 시스템은 유연한 분양가 운영을 위해 다단계 우선순위 로직을 사용합니다.

### 1. 공급 가격 (Contract Price)

시스템은 계약서 생성 시 **[개별 계약 가격 > 기준 공급 가격 > 예산 단가 > 타입 단가]** 순으로 유효한 값을 찾아 적용합니다. 따라서 상세한 공급 가격을 등록하기 전이라도 예산이나 타입 정보만 있다면
대략적인 분양가 산출이 가능합니다.

### 2. 회차별 납부 금액 (Payment Amount)

각 납부 회차의 금액은 설정된 **적용 범위**에 따라 다음 우선순위로 결정됩니다.

|  순위   | 항목 명칭         | 설정 메뉴      | **적용 범위 (Scope)** | 비고                             |
|:-----:|:--------------|:-----------|:------------------|:-------------------------------|
| **1** | **회차별 고정 금액** | [납부 회차 등록] | **프로젝트 전체**       | 모든 타입/동/층에 동일 정액 적용 (예: 업무대행비) |
| **2** | **특별 약정 금액**  | [공급 가격 등록] | **특정 가격 조건**      | 특정 차수+타입+층별 조건에만 예외적 정액 적용     |
| **3** | **타입별 계약 금액** | [계약 금액 등록] | **특정 타입별**        | 특정 타입 전체에 동일한 정액 계약금 적용        |
| **4** | **납부 비율 계산**  | [납부 회차 등록] | **프로젝트 전체**       | (기본값) 공급 가격에 설정된 비율(%)을 곱하여 산출 |

#### 💡 왜 이렇게 복잡하게 구분하나요?

다양한 분양 전략을 유연하게 지원하기 위해서입니다.

- **예시 1**: 모든 세대의 '업무대행비'는 동일하게 1,000만 원으로 고정하고 싶은 경우 -> **회차별 고정 금액(1순위)** 사용
- **예시 2**: 대부분 10% 비율제지만, '펜트하우스'만 계약금을 5,000만 원 정액으로 받고 싶은 경우 -> **특별 약정 금액(2순위)** 사용
- **예시 3**: 동호 지정 전이라 층별 가격이 확정되지 않았지만, '84타입'은 무조건 3,000만 원을 계약금으로 받는 경우 -> **타입별 계약 금액(3순위)** 사용
- **일반적인 경우**: [납부 회차 등록]에서 **납부 비율(4순위)**만 설정하면 모든 유닛의 가격에 맞춰 자동으로 금액이 산출됩니다.
- **잔금 산출**: 잔금은 [총 공급가 - (계약금 + 중도금 + 기타 회차 합계)]로 자동 계산되어 단수 차액을 조정합니다.
