# 프로젝트 등록

프로젝트 등록 페이지는 선택한 프로젝트의 기본 정보를 조회하고,
프로젝트를 신규 등록하거나 수정할 수 있는 화면입니다.

## 주요 기능

- 프로젝트 기본 정보 조회
- 신규 프로젝트 등록
- 프로젝트 정보 수정

## 화면 구성

### 1. 프로젝트 선택

프로젝트를 선택할 수 있습니다.

### 2. 프로젝트 개요

선택한 프로젝트의 주요 정보를 표시합니다.

### 3. 작업 버튼

- 수정하기 : 선택한 프로젝트의 등록 정보를 수정 폼으로 이동
- 등록하기 : 신규 프로젝트의 정보 등록 폼으로 이동

### 프로젝트 입력 폼

프로젝트 등록 및 수정 시 사용하는 입력 영역입니다. 각 폼 항목에 대해서 아래사항을 참고하여 입력합니다.

| 라벨       | 필수입력 여부 | 비고                                                   |
|----------|---------|------------------------------------------------------|
| 업무 프로젝트  | 필수      | 업무시스템 등록 프로젝트 선택, 미등록시 사전 등록 필요                      |
| 프로젝트명    | 필수      | 프로젝트 명칭 입력                                           |
| 정렬순서     |         | 미입력 시 등록 순서 기준, 프로젝트 순서 정렬 필요 시 정렬 순서 숫자 입력          |
| 프로젝트종류   | 필수      | 개발상품 선택, 공동주택, 주상복합, 근린생활시설, 생활형숙박시설, 지식산업센터, 기타     |
| 사업개시년도   | 필수      | 사업개시년도 입력                                            |
| 직영운영여부   |         | 직접 시행 또는 시행 대행(PM) 여부, 직접 시행인 경우 선택                  |
| 동호지정여부   |         | 분양 또는 모집 시 동호지정 여부, 조합원 등 모집시 동호마지정인 경우는 선택하지 않음     |
| 토지환지여부   |         | 사업부지가 환지방식 도시개발사업구역 내 위치하는 경우 선택                     |
| 월별집계시작일  | 필수      | 캐시플로우 상 월별 집계 시작월을 지정. 지정일 이전 자급집행 현황(캐시플로우)은 합산 표시됨 |
| 착공(예상)일  | 필수      | 캐시플로우 상 공사기간 시작일 표시                                  |
| 공사기간     | 필수      | 캐시플로우 상 공가 소요기간 입력                                   |
| 대지위치     |         | 사업부지의 대표지번 등 위치 기재                                   |
| 용도지역지구   | 필수      | 용도지역지구 기재                                            |
| 건축규모     | 필수      | 건축개요 상 건축규모 기재                                       |
| 세대(호/실)수 |         | 건축개요 상 세대수 또는 호/실수 기재                                |
| 대지매입면적   |         | 매입대지면적을 제곱미터 단위로 기재                                  |
| 계획대지면적   |         | 건축개요 상 계획대지면적을 제곱미터 단위로 기재                           |
| 기부체납면적   |         | 건축개요 상 기부채납면적을 제곱미터 단위로 기재                           |
| 지상연면적    |         | 건축개요 상 지상연면적 기재                                      |
| 지하연면적    |         | 건축개요 상 지하연면적 기재                                      |
| 총 연면적    |         | 건축개요 상 총 연면적 기재                                      |
| 건축면적     |         | 건축개요 상 건축면적 기재                                       |
| 용적율(%)   |         | 용적율 기재                                               |
| 건폐율(%)   |         | 건폐율 기재                                               |
| 법정주차대수   |         | 법정주차대수 기재                                            |
| 계획주차대수   |         | 계획주차대수 기재                                            |
