# 차수 타입 설정

프로젝트의 계약 차수, 세부 타입(평형), 그리고 분양가 차등 적용을 위한 층별 범위를 관리합니다.

## 차수 분류 등록

프로젝트의 계약 진행 단계에 따른 차수 그룹을 등록합니다. 이 정보는 계약 등록 및 분양가 산정 시 기준으로 활용됩니다.

- **위치**: `프로젝트 설정 > 차수 타입 설정 > 차수 분류 등록`

#### 주요 입력 항목

| 라벨         | 필수입력 여부 | 비고                                          |
|:-----------|:-------:|:--------------------------------------------|
| 등록차수       |   필수    | 해당 차수의 순서(회차)를 숫자로 입력                       |
| 차수구분       |   필수    | 조합모집 또는 일반분양 중 선택                           |
| 차수그룹명      |   필수    | 차수를 식별할 수 있는 명칭 (예: 1차 조합원, 일반분양분 등)        |
| 미계약세대 기본설정 |         | 미계약 세대 공급가격 생성 시 기본 적용 차수 (프로젝트당 1개만 설정 가능) |

## 타입 정보 등록

공급 시설의 타입(평형)별 세부 정보를 등록하고 면적 및 가격 설정 방식을 정의합니다.

- **위치**: `프로젝트 설정 > 차수 타입 설정 > 타입 정보 등록`

#### 주요 입력 항목

| 라벨        | 필수입력 여부 | 비고                                 |
|:----------|:-------:|:-----------------------------------|
| 타입종류      |   필수    | 공동주택, 오피스텔, 숙박시설 등 시설 용도 선택        |
| 타입구분      |   필수    | 메인유닛 또는 보조시설 여부 선택                 |
| 타입명칭      |   필수    | 타입을 식별하는 명칭 (예: 84A, 59B 등)        |
| 타입색상      |   필수    | 현황판 등에서 타입을 구분하기 위한 전용 색상 지정       |
| 전용면적      |         | 전용면적 (제곱미터)                       |
| 공급면적      |         | 공급면적 (제곱미터)                       |
| 계약면적      |         | 계약면적 (제곱미터)                       |
| 평균가격      |         | 해당 타입의 기준 평균 분양가격                  |
| 공급가 설정 옵션 |   필수    | 가격 책정 기준 (타입별 / 층타입별 / 호별 설정 중 선택) |
| 세대수       |   필수    | 해당 타입으로 공급되는 총 세대수                 |

## 층별 조건 등록

층별로 분양가를 다르게 적용(층별 차등가)하기 위한 층 범위를 설정합니다. 이 설정은 타입 정보 등록에서 '층타입별 설정'을 선택한 경우에 적용됩니다.

- **위치**: `프로젝트 설정 > 차수 타입 설정 > 층별 조건 등록`

#### 주요 입력 항목

| 라벨        | 필수입력 여부 | 비고                                  |
|:----------|:-------:|:------------------------------------|
| 타입종류      |   필수    | 층 조건을 적용할 시설 용도 선택                  |
| 시작 층      |   필수    | 범위의 시작 층 (지하층은 -1 이하 숫자로 입력)        |
| 종료 층      |   필수    | 범위의 종료 층                            |
| 방향/위치(옵션) |         | 향(동/서/남/북)이나 라인 등 추가 구분 조건 입력       |
| 층별 범위 명칭  |   필수    | 해당 범위를 식별할 명칭 (예: 1~3층, 저층부, 기준층 등) |
