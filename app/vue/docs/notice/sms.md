# SMS 발송 관리

계약자에게 개별 또는 단체로 SMS, LMS, MMS 및 카카오 알림톡을 발송하고 내역을 관리하는 페이지입니다.

- **위치**: `고객 고지 관리 > SMS 발송 관리`

## 개요

고객 안내, 납부 독촉, 공지 사항 등을 다양한 채널을 통해 신속하게 전달합니다. 단순 메시지뿐만 아니라 템플릿 변수를 활용한 개인화 메시지, 이미지 첨부(MMS), 예약 발송 기능을 지원하며 모든 발송 내역은 실시간으로 기록됩니다.

## 주요 기능

- **멀티 채널 발송**: 단문(SMS), 장문(LMS), 멀티미디어(MMS), 카카오 알림톡 지원
- **수신자 관리**: 개별 입력, 프로젝트 그룹(전체 계약자) 선택, 엑셀 파일을 통한 대량 업로드
- **개인화 메시지**: `{성함}`, `{미납액}` 등 변수를 사용한 수신자별 맞춤 내용 발송
- **템플릿 관리**: 자주 사용하는 문구를 템플릿으로 저장 및 관리
- **발송 예약**: 즉시 발송 외에 원하는 시점(최소 15분 후 ~ 최대 1개월 이내)으로 예약 설정
- **잔액 및 이력 관리**: 실시간 충전 잔액 확인 및 상세 발송 결과 조회

## 화면 구성

### 1. 메시지 작성 (좌측 영역)

발송할 채널을 선택하고 메시지 내용을 구성합니다.

#### 주요 설정 항목

| 항목 | 설명 |
| :--- | :--- |
| **메시지 타입** | SMS(90자 이내), LMS(2,000자 이내), MMS(이미지 포함) 중 선택 |
| **메시지 템플릿** | 등록된 템플릿을 불러오거나 [관리] 버튼을 통해 새 템플릿 등록 |
| **이미지 첨부** | MMS 선택 시 활성화. JPG 형식, 100KB 미만의 이미지만 가능 (최대 3장) |
| **발송자 번호** | 사전에 승인된 발신 번호 선택 (iwinv 콘솔 사전 등록 필수) |
| **특수문자** | 메시지 내에 삽입할 다양한 기호 및 특수 숫자 제공 |

### 2. 수신자 관리 (우측 영역)

메시지를 받을 대상을 지정합니다. 최대 1,000명까지 일괄 처리가 가능합니다.

#### 수신자 추가 방법

- **개별 번호 입력**: 직접 입력하거나 메모장의 번호를 복사해서 붙여넣기 (줄바꿈 구분)
- **그룹 선택**: 현재 프로젝트의 '전체 계약자' 연락처를 한 번에 불러오기
- **Excel 파일 업로드**: 엑셀의 A열(전화번호)을 기준으로 대량 등록. 템플릿 변수 사용 시 B열 이후의 데이터를 변수값으로 매핑

### 3. 발송 설정 및 실행

| 항목 | 설명 |
| :--- | :--- |
| **발송 시점** | [즉시 발송] 또는 [예약 발송] 선택 |
| **예약 설정** | 예약 발송 시 날짜와 시간을 분 단위로 지정 |
| **잔액 확인** | 현재 가용 잔액을 표시하며, 부족 시 [새로고침] 및 충전 링크 제공 |

### 4. 발송 내역 (탭)

과거에 발송한 모든 기록을 조회합니다.

- **필터링**: 기간, 메시지 타입, 발신번호 등으로 검색 지원
- **상세 보기**: 특정 발송 건의 [확인] 버튼 클릭 시 수신번호 목록과 실제 발송된 내용 확인 가능

## 💡 유의 사항 및 팁

- **MMS 이미지 제한**: 시스템 안정성을 위해 이미지 크기는 **100KB 미만의 JPG** 파일로 엄격히 제한됩니다. 고해상도 사진은 업로드 전 크기 조절이 필요합니다.
- **자동 타입 전환**: 메시지 내용이 90자(한글 약 45자)를 초과하면 시스템이 자동으로 SMS에서 LMS로 타입을 전환합니다.
- **발신번호 사전 등록**: 시스템 보안 정책상 발신번호는 반드시 외부 문자 서비스 제공사(iwinv)의 관리자 페이지에서 먼저 인증을 마쳐야만 본 시스템에 등록하여 사용할 수 있습니다.
- **변수 사용 시 엑셀 형식**: 메시지 템플릿에 `{이름}`과 같은 변수가 있다면, 엑셀 파일의 첫 줄(헤더)에 동일한 변수명이 있어야 정확하게 데이터가 치환됩니다.
