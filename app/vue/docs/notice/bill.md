# 수납 고지서 관리

프로젝트의 납부 회차별 대금 수납 고지서를 생성하고 발행 정보를 관리하는 페이지입니다.

- **위치**: `고객 고지 관리 > 수납 고지서 출력`

## 개요

특정 납부 회차(계약금, 중도금 등)에 대한 고지서를 발행하기 위해 시행사 정보, 수납 계좌, 안내 문구 등을 설정합니다. 설정된 정보를 바탕으로 계약자별 납부 현황(완납/미납)을 확인하고, 선택한 계약자들에 대한 고지서를 PDF 파일로 일괄 생성하여 다운로드할 수 있습니다.

## 주요 기능

- 고지서 기본 정보(발행자, 연락처, 주소 등) 및 수납 계좌 설정
- 발행 대상 회차 및 해당 회차의 납부 기한 지정
- 차수, 타입, 동호수별 계약자 필터링 및 검색
- 회차별 납입 상태(완납/미납) 시각적 확인
- 선택 건별 고지서 PDF 일괄 생성 및 다운로드
- 고지서 출력 시 가격 정보 또는 연체 정보 숨김 옵션 제공

## 화면 구성

### 1. 고지서 발행 정보 설정

고지서의 상단과 하단에 표시될 공식 정보와 수납을 위한 금융 정보를 입력합니다.

#### 주요 입력 항목

| 라벨 | 필수입력 여부 | 비고 |
| :--- | :---: | :--- |
| 발행일자 | 필수 | 고지서 상단에 표시될 공식 발행 날짜 |
| 발행회차 | 필수 | 현재 고지서를 발행할 대상 납부 회차 선택 |
| 당회 납부기한 | | 해당 회차의 입금 마감 기일 (회차 정보에 자동 연동 가능) |
| 시행자/대행사 정보 | 필수 | 고지 주체의 명칭 및 공식 연락처 |
| 수납은행(1, 2) | 필수 | 분양 대금을 수납할 은행, 계좌번호 및 예금주 (최대 2개) |
| 시행자 주소 | 필수 | 고지서에 표시될 시행사(또는 사업단)의 공식 주소 |
| 고지서 제목/내용 | 필수 | 고지서 중앙에 표시될 안내 제목과 상세 공지 문구 |

> **참고**: 고지서를 출력하기 위해서는 반드시 이 설정 데이터를 먼저 [저장]해야 합니다.

### 2. 목록 제어 및 필터링

고지서를 발송할 대상 계약자를 찾기 위한 조건을 설정합니다.

- **필터 항목**: 차수, 타입, 동 선택 및 계약자명/일련번호 검색 지원
- **회차 정보 표시**: 현재 발행 설정된 회차 명칭이 상단에 강조 표시되어 대상을 혼동하지 않도록 돕습니다.

### 3. 고지서 다운로드 옵션

선택된 계약자들에 대한 PDF 생성 조건을 지정합니다.

- **가격정보 미표시**: 체크 시 고지서에서 상세 분양가 및 납부 가액 정보를 숨깁니다.
- **연체정보 미표시**: 체크 시 미납으로 인한 연체료 계산 정보를 제외하고 출력합니다.

### 4. 계약 목록 및 선택

계약자별 현재 수납 상태를 확인하고 고지서를 발행할 대상을 선택합니다.

- **납입상태 표시**: 설정된 '발행회차'를 기준으로 해당 회차까지 돈을 모두 냈는지(**완납중**) 아니면 부족한지(**미납중**)를 자동으로 판별하여 표시합니다.
- **선택 제어**: 이미 완납된 계약자는 고지서 발행 대상에서 제외되도록 체크박스가 비활성화됩니다. (전체 선택 시에도 미납자만 선택됨)

## 💡 업무 팁

- **회차 기한 업데이트**: [고지서 발행 정보 설정]에서 '당회 납부기한'을 수정하고 저장하면, 연결된 [납부 회차 등록]의 약정일 데이터도 함께 업데이트되어 관리가 편리합니다.
- **테스트 출력**: 대량 발행 전 1~2건의 계약자만 선택하여 PDF 레이아웃과 오타 여부를 먼저 확인하는 것이 좋습니다.
