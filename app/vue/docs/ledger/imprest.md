# 운영비 내역 관리

프로젝트 수행을 위해 별도로 운영되는 운영비(전도금) 계좌의 입출금 내역을 기록하고, 이를 회계 계정 및 프로젝트별 계약 정보와 연결하여 관리하는 페이지입니다.

- **위치**: `회계 자금 관리 > 운영비 내역 관리`

## 개요

본사 관리 계좌 외에 프로젝트 현장에서 사용하는 운영비 계좌(이하 운영 계좌)의 실제 거래 데이터를 등록합니다. 등록된 거래는 회계상 적절한 계정 과목으로 분류되어야 하며, 특히 외주비나 사업비 집행 시 관련 계약 정보와 매칭하여 프로젝트 비용 집행의 투명성을 확보합니다.

## 주요 기능

- **운영비 특화 필터링**: 프로젝트별 운영 계좌(Imprest Account)를 기준으로 기간별, 계정별 내역 조회
- **다수 거래 일괄 등록**: 신규 등록 시 여러 건의 독립된 은행 거래를 한 화면에서 동시에 입력 가능
- **인라인 편집 (Inline Editing)**: 목록 화면에서 항목을 더블 클릭하여 적요, 메모, 금액, 계정 등을 즉시 수정
- **이체 수수료 자동 입력**: 출금 항목 옆의 아이콘 클릭 시 이체 수수료(500원) 분개를 자동으로 생성
- **대체 거래 자동 감지**: 대체(출금) 계정 선택 시 상대 계좌의 대체(입금) 거래건을 자동으로 추가 제안
- **엑셀 연동 및 템플릿**: 회계 분류 내역의 엑셀 템플릿 다운로드 및 대량 업로드 지원
- **하이라이트 이동**: 외부 메뉴(예: 자금 일보)에서 유입 시 해당 거래를 즉시 찾아 스크롤 및 강조 표시

## 화면 구성

### 1. 목록 제어 및 필터링
특정 프로젝트의 운영비 내역을 조건별로 검색하는 영역입니다.

- **검색 조건**: 거래일자 범위, 입출금 구분, 계정 분류, 운영 계좌 선택, 관련 계약 정보 및 검색어(적요/거래처)
- **조회 결과**: 하단에 조건에 부합하는 전체 거래 건수가 표시되어 데이터 규모를 즉시 확인 가능합니다.

### 2. 운영비 입출금 목록
조회된 내역을 테이블 형태로 표시하며, 각 행은 '은행거래내역'과 '분류 내역'으로 구성됩니다.

- **은행거래내역**: 실제 통장에 기록된 날짜, 계좌 별칭, 적요, 입/출금액 및 확인 상태 표시
- **분류 내역 (분개)**: 해당 거래의 회계 계정, 거래처, 분류 금액, 지출 증빙 정보. 계약 정보가 연결된 경우 아이콘(📖, 👤)으로 표시됩니다.
- **금액 일치 검증**: 거래 금액과 분류 금액의 합계가 일치하면 ✅ 아이콘이 표시됩니다. 불일치 시 차액 정보가 노란색 경고 행으로 표시됩니다.

### 3. 거래 등록 및 수정
새로운 내역을 생성하거나 기존 내역을 수정하는 화면입니다.

- **신규 등록 (다수 건)**: [새 거래 등록] 버튼을 통해 여러 건의 은행 거래를 동시에 추가할 수 있는 행 기반 입력 폼을 제공합니다.
- **상세 수정**: 개별 행의 수정 아이콘을 클릭하여 상세 수정 페이지로 진입하거나, 목록에서 더블 클릭하여 필요한 필드만 인라인으로 수정할 수 있습니다.

## 💡 주요 관리 포인트

### 인라인 편집과 데이터 반영
목록에서 더블 클릭을 통해 즉시 수정할 수 있습니다. 특히 **계정 과목** 수정 시, 해당 계정이 계약 정보나 계약자 매칭이 필수인 경우 자동으로 선택 팝업이 호출되어 데이터의 정합성을 강제합니다.

### 과거 데이터 수정 제한
데이터 무결성 보호를 위해 **최종 정산 일자** 이전의 데이터나, 등록 후 일정 기간(기본 10일)이 경과한 과거 데이터는 일반 권한으로 수정이 제한됩니다. 수정이 필요한 경우 상급 권한(SuperAuth)을 가진 관리자에게 요청해야 합니다.

### 대체 거래(자금 이체) 처리
운영 계좌 간 자금 이동이나 본사 계좌에서 운영 계좌로 자금을 넘겨받는 경우, 출금 측에서 '대체(출금)' 계정을 선택하면 시스템이 입금 측의 '대체(입금)' 거래를 자동으로 생성할지 확인합니다. 이를 통해 이중 입력의 번거로움을 줄이고 자금 흐름을 일치시킵니다.
