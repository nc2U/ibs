# 실행기록 (Activity)

**실행기록** 페이지는 프로젝트 내에서 발생한 모든 변경 사항과 활동 내역을 시간 순으로 확인할 수 있는 '프로젝트 히스토리' 화면입니다. 누가, 언제, 어떤 업무를 수행했는지 한눈에 파악할 수 있습니다.

## 1. 화면 개요
실행기록은 프로젝트의 살아있는 기록부와 같습니다. 업무의 생성, 수정, 댓글 작성뿐만 아니라 공지사항이나 문서의 업데이트 내역까지 날짜별로 정리되어 표시됩니다. 기본적으로 최근 10일간의 기록을 먼저 보여줍니다.

## 2. 주요 기능 안내

### 📅 기간별 기록 조회 및 이동
- **기간 확인**: 목록 상단에 표시된 날짜 범위를 통해 현재 보고 있는 기록의 기간을 확인할 수 있습니다.
- **날짜 이동**: 목록 하단의 **[« 뒤로]** 버튼은 10일 이전의 기록으로, **[다음 »]** 버튼은 10일 이후의 기록으로 이동합니다.
- **특정 날짜 지정**: 우측 설정창의 '10일 기록' 달력 아이콘을 클릭하여 특정 날짜를 선택하면, 해당 날짜를 기준으로 과거 10일간의 기록을 즉시 불러옵니다.

### 👥 사용자별 활동 필터링
- 특정 팀원의 활동만 모아보고 싶을 때 우측 설정창의 **[사용자]** 필터를 사용합니다.
- 본인의 활동 내역만 확인하려면 목록에서 **[<< 나 >>]**를 선택하고 '적용' 버튼을 클릭하세요.

### 🗂️ 활동 유형별 필터링
보고 싶은 활동의 종류만 선택하여 목록을 간소화할 수 있습니다.
- **업무**: 업무 생성, 상태 변경, 댓글 작성 등 이슈 관련 모든 활동.
- **공지**: 새로운 공지사항 등록 및 수정 내역.
- **문서**: 관련 문서 자료의 업로드 및 관리 내역.
- **글**: 게시판이나 포럼 등에 작성된 게시글 내역.

### 🌲 하위 프로젝트 포함
- 현재 프로젝트에 포함된 하위 프로젝트들의 활동까지 함께 보고 싶다면 **[하위 프로젝트]** 체크박스를 선택하세요. 대규모 프로젝트의 전체 흐름을 파악할 때 유용합니다.

## 3. 이용 팁
- **상세 내용 바로가기**: 활동 기록에 표시된 업무 제목이나 게시글 제목을 클릭하면 해당 항목의 상세 페이지로 직접 이동하여 자세한 내용을 확인할 수 있습니다.
- **히스토리 추적**: 특정 업무의 상태가 왜 변경되었는지 궁금할 때 해당 날짜의 실행기록을 찾아보면 전후 맥락을 파악하는 데 도움이 됩니다.
