# 업무 프로젝트

업무 관리 시스템의 메인 화면인 **프로젝트 목록** 페이지에 대한 사용자 가이드입니다. 이 페이지에서는 진행 중인 다양한 업무 프로젝트를 확인하고 관리할 수 있습니다.

## 1. 화면 개요
프로젝트 목록 화면은 사용자가 참여 중이거나 열람 가능한 모든 프로젝트를 카드 형태로 보여줍니다. 상단의 검색 도구와 하단의 상태 안내를 통해 효율적인 프로젝트 관리가 가능합니다.

## 2. 주요 기능 안내

### 🔍 프로젝트 검색 및 필터링
- 화면 상단의 검색 바를 이용해 프로젝트 이름으로 원하는 프로젝트를 빠르게 찾을 수 있습니다.
- 특정 조건(상태, 유형 등)을 선택하여 목록을 필터링할 수 있어, 수많은 프로젝트 중 필요한 정보만 선별하여 조회할 수 있습니다.

### ➕ 새 프로젝트 생성 (관리자 전용)
- **프로젝트 관리 권한**(`workManager`)을 가진 사용자는 우측 상단의 **[새 프로젝트]** 링크를 통해 새로운 프로젝트를 생성할 수 있습니다.
- 일반 사용자의 경우 해당 버튼이 노출되지 않으며, 할당된 프로젝트 내에서의 활동만 가능합니다.

### 🗂️ 프로젝트 카드 목록
- 각 프로젝트는 하나의 카드로 표현되며, 프로젝트의 핵심 요약 정보를 담고 있습니다.
- 화면 크기(PC, 태블릿, 모바일)에 따라 자동으로 레이아웃이 변경되어 어떤 환경에서도 최적화된 목록을 확인할 수 있습니다.

### 💡 아이콘 및 범례 안내
화면 하단에서는 프로젝트의 상태를 나타내는 아이콘의 의미를 확인할 수 있습니다.
- **내 프로젝트** (<v-icon icon="mdi-account-tag" color="success" size="15" />): 본인이 담당자로 지정되어 있거나 구성원으로 참여 중인 프로젝트를 나타냅니다.
- **내 북마크** (<v-icon icon="mdi-bookmark" color="info" size="15" />): 사용자가 즐겨찾기에 추가하여 별도로 관리 중인 프로젝트를 나타냅니다.

## 3. 이용 팁
- 프로젝트가 많아질 경우 검색 기능을 활용하여 '내 프로젝트'나 '내 북마크' 위주로 필터링하면 업무 효율을 높일 수 있습니다.
- 프로젝트 카드를 클릭하면 해당 프로젝트의 상세 업무(이슈), 추진 현황(로드맵), 캘린더 등 세부 메뉴로 이동할 수 있습니다.
