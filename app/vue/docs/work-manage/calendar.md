# 달력

업무 관리 시스템의 일정을 시각적으로 확인하고 관리할 수 있는 **달력** 페이지에 대한 사용자 가이드입니다.

## 1. 화면 개요
달력 화면은 등록된 모든 업무(Task)의 시작일과 종료일을 월간 단위로 시각화하여 보여줍니다. 전역 관리 화면과 개별 프로젝트 화면에서 각각 용도에 맞게 활용할 수 있습니다.

## 2. 주요 기능 안내

### 📅 전역 달력 vs 프로젝트 달력
- **전역 달력**: 시스템 전체 프로젝트에서 본인이 조회 가능한 모든 업무를 통합하여 보여줍니다. 개인의 전체적인 업무 일정을 파악하기에 좋습니다.
- **프로젝트 달력**: 특정 프로젝트 내의 업무들만 집중적으로 조회합니다. 해당 프로젝트의 마일스톤과 세부 일정을 관리하는 데 유용합니다.

### 🎨 업무 상태별 색상 (테마 대응)
업무 바(Bar)의 색상은 업무의 현재 상태를 나타냅니다. 사용자의 가독성을 위해 테마(Light/Dark)에 따라 최적화된 저채도 컬러 팔레트가 자동으로 적용됩니다.

- <span style="color: #e57373">●</span> **신규**: 즉각적인 확인이 필요한 새로운 업무
- <span style="color: #64b5f6">●</span> **진행**: 현재 수행 중인 활성 업무
- <span style="color: #ffb74d">●</span> **검토**: 결과 보고 후 피드백을 대기 중인 업무
- <span style="color: #90a4ae">●</span> **보류 / 거절**: 일시 중지되거나 처리가 반려된 업무
- <span style="color: #81c784">●</span> **종료**: 성공적으로 완료되어 닫힌 업무

### 🔔 오늘의 일정 알림 아이콘
업무 바의 제목 앞에는 **오늘 날짜**를 기준으로 중요한 상태를 알려주는 아이콘이 표시됩니다.

- <i class="mdi mdi-arrow-right-bold text-success"></i> **오늘 시작**: 오늘부터 일정이 시작되는 업무입니다.
- <i class="mdi mdi-arrow-left-bold text-danger"></i> **오늘 종료**: 오늘까지 반드시 완료해야 하는 기한 업무입니다.
- <i class="mdi mdi-rhombus text-danger"></i> **오늘 시작/종료**: 오늘 하루 동안 시작하고 마무리해야 하는 당일 처리 업무입니다.

### 📊 업무 현황 요약 카드
캘린더 하단에는 현재 화면에 표시된 업무들의 통계 정보가 제공됩니다.
- **전체 건수**: 필터링된 조건 내의 총 업무 수
- **진행중 / 완료**: 상태별 업무 비중을 실시간으로 확인하여 이번 달의 성과를 한눈에 파악할 수 있습니다.

## 3. 이용 팁
- **상세 정보 확인**: 달력의 업무 바를 클릭하면 해당 업무의 상세 페이지로 즉시 이동합니다.
- **직관적인 타이틀**: 각 업무는 `[업무유형] 제목 (담당자)` 형식으로 표시되어 상세 페이지에 들어가지 않고도 핵심 정보를 알 수 있습니다.
- **필터링 활용**: 상단의 검색 도구를 사용해 특정 프로젝트나 담당자별로 일정을 선별하여 조회해 보세요.
- **오늘 표시**: 현재 날짜는 파란색 사각형 배지로 강조되어 있어 전체 일정 중 오늘의 위치를 쉽게 찾을 수 있습니다.
