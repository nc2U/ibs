# 계약 해지 관리

계약 해지 정보를 등록, 조회, 관리하는 페이지입니다.

-   **위치**: `계약 관리 > 계약 해지 관리`

## 개요

특정 계약에 대한 해지 내역을 조회하고, 새로운 해지 건을 등록하거나 기존 해지 정보를 수정할 수 있습니다. 해지 신청일, 환불 금액, 환불 계좌 정보 등 해지 관련 상세 정보를 관리합니다.

## 주요 기능

-   계약자 검색 및 선택 기능 (해지 대상 계약자 지정)
-   계약 해지 현황 목록 조회 및 엑셀 출력
-   새로운 계약 해지 등록
-   기존 계약 해지 정보 수정 및 삭제
-   해지 신청일, 환불 금액, 환불 계좌 정보 등 주요 해지 정보 관리
-   해지 상태 (신청 취소, 해지 신청, 해지 완료, 자격 상실) 설정
-   URL 파라미터를 통한 특정 해지 건 하이라이트 및 페이지 이동 지원

## 화면 구성

### 1. 계약자 검색 및 선택 (`ContController.vue`)

해지 대상 계약자를 검색하고 선택할 수 있는 기능을 제공합니다. 검색 결과는 버튼 형태로 표시되어 쉽게 선택할 수 있습니다.

#### 주요 기능

-   **검색 입력 필드**: 계약자명, 비고, 계약 일련번호로 검색
-   **검색 버튼**: 검색 실행
-   **검색 결과 목록**: 검색된 계약자 목록을 버튼 형태로 표시, 클릭 시 해당 계약자를 해지 대상 계약자로 설정하고 상세 정보 로드

### 2. 계약 관리 내비게이션 (`ContNavigation.vue`)

계약 상세 관리, 권리 의무 승계, 계약 해지 관리 페이지로 이동할 수 있는 탭 형태의 내비게이션을 제공합니다. 현재 활성화된 페이지를 시각적으로 표시합니다.

#### 주요 항목

-   계약 상세 관리 버튼
-   권리 의무 승계 버튼
-   계약 해지 관리 버튼

### 3. 계약자 정보 알림 (`ContractorAlert.vue`)

현재 선택된 해지 대상 계약자의 주요 정보(계약자명, 현재 상태)를 알림 형태로 표시합니다. 계약자 선택 해제 기능과 함께, 승계 진행 중이거나 해지된 계약자에 대한 특별 알림을 제공합니다.

#### 주요 표시 항목

-   계약자명
-   현재 상태 (청약, 계약, 해지신청, 해지완료 등)
-   승계 진행 중 여부
-   계약자 선택 해제 버튼

### 4. 해지 등록/수정 버튼 (`ReleasetButton.vue`)

새로운 해지 건을 등록하거나, 현재 조회 중인 해지 건을 수정하기 위한 버튼을 제공합니다. 해지 진행 상태에 따라 버튼의 텍스트와 색상이 변경됩니다.

#### 주요 기능

-   '등록하기' 또는 '수정하기' 버튼 클릭 시 `계약 해지 등록/수정 폼` 모달 호출.

### 5. 계약 해지 현황 목록 (`ReleaseList.vue`, `Release.vue`)

등록된 계약 해지 건들의 목록을 테이블 형태로 표시합니다. 각 해지 건에 대한 상세 정보와 확인 버튼을 제공합니다.

#### 주요 표시 항목

| 항목 | 설명 |
| :--- | :--- |
| 계약 해지자 | 해지 대상 계약자의 이름 |
| 현재상태 | 해지 상태 (신청 취소, 해지 신청, 해지 완료, 자격 상실) |
| 환불(예정)금액 | 계약 해지로 인한 환불 예정 금액 |
| (환불)은행명 | 환불 계좌의 은행명 |
| (환불)계좌번호 | 환불 계좌 번호 |
| (환불)예금주 | 환불 계좌의 예금주 |
| 해지신청일 | 해지 신청이 접수된 날짜 |
| 환불처리일 | 해지 및 환불 처리가 완료된 날짜 |
| 비고 | 해지 상세 정보를 확인하거나 수정하는 버튼 |

### 6. 계약 해지 등록/수정 폼 (`ReleaseForm.vue`)

새로운 계약 해지 정보를 등록하거나 기존 정보를 수정하는 모달 폼입니다. 해지 대상 계약자 정보는 자동으로 채워지며, 해지 관련 상세 정보를 입력합니다.

#### 주요 입력 항목

-   **계약자**: (자동 입력) 해지 대상 계약자 정보.
-   **구분**: 필수. 해지 상태 (신청 취소, 해지 신청, 해지 완료, 자격 상실) 선택.
-   **환불(예정)금액**: 해지로 인한 환불 예정 금액.
-   **거래은행(환불계좌)**: 환불 계좌의 은행명.
-   **계좌번호(환불계좌)**: 환불 계좌 번호.
-   **예금주(환불계좌)**: 환불 계좌의 예금주.
-   **해지신청일**: 필수. 해지 신청이 접수된 날짜.
-   **해지(환불)처리일**: 해지 및 환불 처리가 완료된 날짜. (해지 완료 상태 시 필수)
-   **비고**: 기타 특이사항.
