# 계약 등록 조회

프로젝트의 청약 및 계약 정보를 등록, 조회, 관리하는 페이지입니다.

- **위치**: `계약 관리 > 계약 등록 및 조회`

## 개요

프로젝트별 청약 및 계약 현황을 한눈에 확인하고, 신규 계약을 등록하거나 기존 계약 정보를 수정 및 조회할 수 있습니다. 다양한 필터링 옵션을 통해 원하는 계약 정보를 효율적으로 검색하고, 엑셀로 출력할 수
있습니다.

## 주요 기능

- 프로젝트별 청약/계약 현황 요약 정보 제공
- 다양한 조건(차수, 타입, 동호수, 계약일 등)으로 계약 목록 필터링 및 검색
- 신규 청약 및 계약 등록
- 기존 계약 정보 수정 및 상세 조회
- 계약 목록 엑셀 출력 (출력 항목 선택 가능)
- URL 파라미터를 통한 특정 계약 하이라이트 및 페이지 이동 지원

## 화면 구성

### 1. 계약 현황 요약 (`ContractSummary.vue`)

선택된 프로젝트의 전체 청약 및 계약 현황을 타입별, 차수별로 요약하여 보여줍니다. 총 세대수, 청약 건수, 계약 건수, 잔여 세대, 계약율, 분양율 등의 주요 지표를 확인할 수 있습니다.

#### 주요 표시 항목

- 프로젝트명
- 타입별 세대수, 청약건수, 계약건수 (차수별 합계 포함)
- 잔여세대(실), 계약율, 분양율(청약+계약)

### 2. 목록 제어 및 필터링 (`ListController.vue`)

계약 목록을 조회하기 위한 다양한 필터링 및 정렬 옵션을 제공합니다.

#### 주요 입력 항목

| 라벨            | 필수입력 여부 | 비고                                  |
|:--------------|:-------:|:------------------------------------|
| 표시 개수         |         | 페이지당 표시할 계약 건수 선택                   |
| 정렬 기준         |         | 등록일시, 계약일자, 일련번호, 계약자명 등으로 정렬       |
| 등록상태          |         | 일반분양, 미인가조합원, 인가조합원, 부적격조합원 등으로 필터링 |
| 현황 구분         |         | 청약 현황 또는 계약 현황 선택                   |
| 차수 선택         |         | 특정 차수 그룹으로 필터링                      |
| 타입 선택         |         | 특정 유닛 타입으로 필터링                      |
| 공급계약          |         | 공급계약 체결 여부로 필터링                     |
| 동 선택          |         | 특정 동으로 필터링                          |
| 동호 미지정        |         | 동호수가 미지정된 계약만 조회                    |
| 계약일 (From/To) |         | 특정 기간 내 계약일로 필터링                    |
| 검색            |         | 계약자, 전화번호, 일련번호, 비고 등으로 검색          |

### 3. 신규 계약 등록 (`AddContract.vue`, `ContractForm.vue`)

새로운 청약 또는 계약 정보를 등록할 수 있는 모달 폼을 제공합니다.

#### 주요 입력 항목 (ContractForm)

| 라벨               |   필수입력 여부    | 비고                                   |
|:-----------------|:------------:|:-------------------------------------|
| 구분               |      필수      | 청약 또는 계약 선택                          |
| 차수               |      필수      | 계약이 속할 차수 그룹 선택                      |
| 타입               |  필수 (계약 시)   | 유닛 타입 선택 (공급가격 정보 미등록 시 경고 표시)       |
| 계약코드             | 필수 (유닛 설정 시) | 유닛 코드 선택                             |
| 동호수              |              | 동호수 선택                               |
| 공급계약             |              | 공급계약 체결 여부 및 체결일자                    |
| 계약일자             |      필수      | 청약일자 또는 계약일자                         |
| 계약자명             |      필수      | 계약자 이름                               |
| 생년월일             |  필수 (계약 시)   | 계약자 생년월일                             |
| 성별               |  필수 (계약 시)   | 계약자 성별                               |
| 휴대전화             |      필수      | 계약자 휴대전화 번호                          |
| 이메일, 집전화, 기타 연락처 |              | 추가 연락처 정보                            |
| 계약금 등록/수정        |              | 계약금 입금 정보 (입금일자, 금액, 계좌, 입금자명, 납부회차) |
| 주민등록 주소          |  필수 (계약 시)   | 계약자 주민등록 주소                          |
| 우편수령 주소          |  필수 (계약 시)   | 우편물 수령 주소 (주민등록 주소와 동일 설정 가능)        |
| 비고               |              | 기타 특이사항                              |
| 계약서 파일           |              | 계약서 스캔본 등 첨부 파일                      |

### 4. 계약 목록 (`ContractList.vue`, `Contract.vue`)

필터링 조건에 따라 조회된 계약 목록을 테이블 형태로 표시합니다. 각 계약 건에 대한 상세 정보와 수정/보기 버튼을 제공합니다.

#### 주요 표시 항목

| 항목     | 설명                           |
|:-------|:-----------------------------|
| 일련번호   | 계약의 고유 일련번호 (상세 보기 링크)       |
| 등록상태   | 계약자의 등록상태 (예: 미인가조합원, 인가조합원) |
| 차수     | 계약이 속한 차수 그룹                 |
| 타입     | 계약된 유닛의 타입                   |
| 계약자    | 계약자명 (상세 보기 링크)              |
| 동호수    | 계약된 유닛의 동호수 (상세 보기 링크)       |
| 가입 계약일 | 청약 또는 가입 계약일                 |
| 공급 계약일 | 공급 계약 체결일                    |
| 공급계약가격 | 계약된 유닛의 공급 계약 가격             |
| 납입금액합계 | 현재까지 납입된 총 금액                |
| 최종납입회차 | 마지막으로 납입된 회차 정보              |
| 계약서    | 첨부된 계약서 파일 다운로드 링크           |
| 비고     | 계약 수정 및 상세 보기 버튼             |

### 5. 엑셀 출력 항목 선택 (`SelectItems.vue`)

계약 목록을 엑셀로 출력할 때 포함할 항목들을 선택할 수 있는 기능을 제공합니다.

#### 주요 선택 항목

일련번호, 등록상태, 차수, 타입, 계약자, 동호수, 가입계약일, 공급계약일, 건물가, 대지가, 부가세, 공급가액, 납입금합계, 생년월일, 연락처, 이메일, 주소(등본/우편), 비고 등.

### 6. 주소 변경 등록 (`AddressForm.vue`)

계약자의 주민등록 주소 및 우편물 수령 주소 이력을 관리하고, 새로운 주소로 변경 등록할 수 있는 모달 폼입니다.

#### 주요 기능

- 새로운 주민등록 주소 및 우편물 수령 주소 입력 (다음 우편번호 API 연동)
- 현재 주소와 과거 주소 이력 조회
- 과거 주소 이력 중 하나를 현재 주소로 다시 변경하는 기능
