# 계약 상세 관리

특정 계약자의 상세 계약 정보를 조회하고 관리하는 페이지입니다.

- **위치**: `계약 정보 관리 > 계약 상세 관리`

## 개요

선택된 계약자의 계약 기본 정보, 금액 정보, 납부 내역, 구비 서류 제출 현황, 상담 내역 등 모든 관련 정보를 한곳에서 확인하고 관리할 수 있습니다. 권리 의무 승계 및 계약 해지 관리 페이지로 이동할 수 있는 내비게이션을 제공합니다.

## 주요 기능

- 계약자 검색 및 선택 기능
- 계약 상세 정보 (기본 정보, 계약자 정보, 주소, 연락처, 메모, 계약서 파일) 조회 및 관리
- 금액 정보 (분양가, 총 납부액, 완납 차수) 및 납부 내역 조회
- 각종 고지/확인서 발급 기능
- 구비 서류 제출 현황 관리 (증명서류, 동의서류)
- 상담 내역 등록, 조회, 수정, 삭제 기능
- 권리 의무 승계 및 계약 해지 관리 페이지로의 이동

## 화면 구성

### 1. 계약자 검색 및 선택

계약자명, 비고, 계약 일련번호 등으로 계약자를 검색하고 선택할 수 있는 기능을 제공합니다. 검색 결과는 버튼 형태로 표시되어 쉽게 선택할 수 있습니다.

#### 주요 기능

- **검색 입력 필드**: 계약자명, 비고, 계약 일련번호로 검색
- **검색 버튼**: 검색 실행
- **검색 결과 목록**: 검색된 계약자 목록을 버튼 형태로 표시, 클릭 시 해당 계약자의 상세 정보 로드

### 2. 계약 관리 내비게이션

계약 상세 관리, 권리 의무 승계, 계약 해지 관리 페이지로 이동할 수 있는 탭 형태의 내비게이션을 제공합니다. 현재 활성화된 페이지를 시각적으로 표시합니다.

#### 주요 항목

- 계약 상세 관리 버튼
- 권리 의무 승계 버튼
- 계약 해지 관리 버튼

### 3. 계약자 정보 알림

현재 선택된 계약자의 주요 정보(계약자명, 현재 상태)를 알림 형태로 표시합니다. 계약자 선택 해제 기능과 함께, 승계 진행 중이거나 해지된 계약자에 대한 특별 알림을 제공합니다.

#### 주요 표시 항목

- 계약자명
- 현재 상태 (청약, 계약, 해지신청, 해지완료 등)
- 승계 진행 중 여부
- 계약자 선택 해제 버튼

### 4. 계약 상세 정보

선택된 계약자의 모든 상세 정보를 표시하는 메인 영역입니다. '계약관련 내역', '납부관련 내역', '기타추가 정보' 세 가지 탭으로 구성되며, 사용자가 원하는 탭을 선택하여 정보를 확인할 수 있습니다. 여러 탭을 동시에 열어 볼 수도 있습니다.

#### 주요 구성 요소

- **탭 선택 버튼**: 계약관련 내역, 납부관련 내역, 기타추가 정보 탭을 선택하거나 동시에 열 수 있는 토글 버튼 그룹.
- **목록으로 버튼**: 계약 목록 페이지로 돌아가는 버튼.

#### 4.1. 계약관련 내역

계약의 기본 정보와 계약자의 상세 정보를 표시합니다.

##### 주요 표시 항목

- **계약 기본 정보**: 계약번호, 분양차수, 타입, 호수, 계약일, 공급계약일, 활성화 여부, 공급계약 완료 여부.
- **계약자 상세 정보**:
    - **기본 정보**: 이름, 생년월일, 성별, 자격구분, 상태, 청약일.
    - **연락처**: 휴대폰, 집전화, 기타전화, 이메일.
    - **주소 정보**: 주민등록 주소, 우편물 수령 주소 (탭 전환 가능), 주소 변경 등록 기능, 과거 주소 내역 조회.
    - **메모**: 계약자에 대한 기타 특이사항.
- **계약서 파일**: 첨부된 계약서 파일 목록, 다운로드, 수정, 삭제, 신규 업로드 기능.

#### 4.2. 납부관련 내역

계약과 관련된 금액 정보 및 납부 내역을 표시하고, 각종 고지서 및 확인서를 발급할 수 있는 기능을 제공합니다.

##### 주요 표시 항목

- **금액 정보**: 분양가 (건물가, 토지가, 부가세 상세 포함), 총 납부액, 완납 차수.
- **납부 내역**: 각 납부 건의 납부일, 회차, 금액. (건별 납부 관리 페이지로 이동 링크 포함)
- **고지 / 확인서 발급**: 대금수납 고지서, 납부내역 확인서 (일반/확인), 할인/가산금 내역서, 일자별 연체료, 주택 인도 증서 등 다양한 문서 발급 기능.

#### 4.3. 기타추가 정보

계약과 관련된 구비 서류 제출 현황 및 상담 내역을 관리합니다. 각 섹션은 확장/축소 가능하며, 로컬 스토리지에 상태가 저장됩니다.

##### 주요 구성 요소

- **구비서류 제출 현황**:
    - **설명**: 계약자가 제출해야 할 구비 서류 목록과 제출 현황을 관리합니다. 증명서류와 동의서류 탭으로 구분됩니다.
    - **주요 기능**:
        - 필수 서류 미제출 건수 및 완료율 표시.
        - 서류명, 필수 여부, 필요 수량, 제출 수량 표시.
        - 제출 수량 편집 및 자동 저장.
        - 첨부 파일 업로드, 다운로드, 삭제 기능.
        - 등록된 필요 서류가 없을 경우, 구비 서류 등록 페이지로 이동 링크 제공.
- **상담 내역**:
    - **설명**: 계약자와의 상담 내역을 기록하고 관리합니다.
    - **주요 기능**:
        - 상담 내역 등록 폼 (상담일자, 채널, 유형, 제목, 내용, 처리상태, 중요도, 후속조치 등).
        - 상담 내역 필터링 (전체, 처리대기, 처리중, 처리완료, 보류).
        - 상담 내역 목록 조회 (상담일자, 채널, 유형, 제목, 처리상태, 작업).
        - 각 상담 내역의 상세 보기, 수정, 삭제 기능.
        - 페이지네이션.
