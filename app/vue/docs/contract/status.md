# 동호 배치 현황

프로젝트의 동호수별 계약 현황을 시각적으로 확인하고, 전체적인 청약 및 계약 요약 정보를 제공하는 페이지입니다.

-   **위치**: `계약 관리 > 동호 배치 현황`

## 개요

각 동(건물)별, 층별, 라인별 유닛의 현재 상태(공실, 청약, 계약, 홀딩)를 색상으로 구분하여 직관적으로 파악할 수 있습니다. 전체 프로젝트의 청약 및 계약 현황 요약 정보와 함께, 계약자명 표시 여부 및 엑셀 출력 기능을 제공합니다.

## 주요 기능

-   프로젝트별 청약 및 계약 현황 요약 정보 제공
-   동호수 배치도를 통한 유닛별 상태 시각화 (공실, 청약, 계약, 홀딩)
-   동(건물)별 유닛 배치도 동적 로딩 (Lazy Loading)
-   계약자명 표시/숨김 토글 기능
-   동호수 현황표 엑셀 출력 기능
-   유닛 클릭 시 해당 계약자의 상세 정보로 이동 (계약자명 표시 시)

## 화면 구성

### 1. 계약 현황 요약 (`ContSummary.vue`)

현재 프로젝트의 전체적인 청약 및 계약 현황을 요약하여 보여줍니다.

#### 주요 표시 항목

-   총 세대수
-   청약 세대수
-   계약 세대수
-   동호 미배정 세대수
-   홀딩 세대수
-   합계 (청약 + 계약 세대수)
-   잔여 세대수
-   타입별 색상 범례

### 2. 동호수 현황표 (`TableTitleRow.vue`, `ContractBoard.vue`)

프로젝트의 각 동(건물)별 동호수 배치도를 시각적으로 표시합니다. 각 유닛은 현재 상태에 따라 다른 색상으로 표시됩니다.

#### 주요 기능

-   **엑셀 출력 버튼**: 현재 동호수 현황표를 엑셀 파일로 다운로드합니다.
-   **계약자명 표시 토글**: 동호수 배치도에 계약자명을 표시하거나 숨길 수 있습니다.
-   **동(건물)별 배치도 (`LazyBuilding.vue`, `Building.vue`)**:
    -   각 동(건물)의 층별, 라인별 유닛을 그리드 형태로 표시합니다.
    -   `LazyBuilding.vue`를 통해 화면에 보이는 동만 로딩하여 성능을 최적화합니다.
    -   **유닛 (`Unit.vue`)**:
        -   각 개별 유닛의 상태를 색상으로 표시합니다.
        -   **색상 구분**:
            -   **공실**: 배경색 없음 (또는 기본색)
            -   **청약**: 연한 노란색 계열
            -   **계약**: 연한 회색 계열
            -   **홀딩**: 진한 회색 계열
            -   **타입별 색상**: 유닛 타입에 따라 다른 색상으로 유닛명 배경 표시
        -   **표시 정보**: 유닛명, (옵션) 계약자명, 홀딩 사유 (툴팁)
        -   **기능**: 계약자명 클릭 시 해당 계약자의 상세 정보 페이지로 이동.
        -   **피로티**: 피로티 층은 별도의 배경색으로 표시.
