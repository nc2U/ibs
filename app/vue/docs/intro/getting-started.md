# 시작하기

> IBS Platform은 부동산 개발 프로젝트의 자금, 계약, 고객, 문서, 소송 데이터를 통합 관리하는 시스템입니다.
> 이 문서는 신규 사용자가 IBS Platform에 접속하고 기본적인 업무를 시작할 수 있도록 안내합니다.

## 이 문서의 대상

이 문서는 다음 사용자를 대상으로 합니다.

- 신규 입사자
- 프로젝트 담당자
- 회계 및 자금 담당자
- 시스템 관리자

## IBS Platform 한눈에 보기

### IBS의 역할

IBS Platform은 부동산 개발사업의 전 과정에서 발생하는 데이터를 하나의 시스템에서 통합 관리합니다.

```aiignore
┌────────────────────────────────────────────┐
│ 프로젝트 관리 - 등록 / 설정 / 부지 관리       │
└────────────────────────────────────────────┘
                     ↓
┌────────────────────────────────────────────┐
│ 계약 정보 관리 - 등록 / 상세 / 승계 / 해지    │
└────────────────────────────────────────────┘
                     ↓
┌────────────────────────────────────────────┐
│ 계약 납부 관리 - 고지서 / SMS                │
└────────────────────────────────────────────┘
                     ↓
┌────────────────────────────────────────────┐
│ 회계 자금 관리 - 정산 / 거래 / 운영비 관리    │
└────────────────────────────────────────────┘
                     ↓
┌───────────────────────────────────────────┐
│ 문서 소송 관리 - 일반 / 소송 문서 / 사건 관리│
└───────────────────────────────────────────┘
```

- 프로젝트 기본 정보 설정

- 고객 및 수납 관리
- 일반 회계 및 자금 관리
- 예산 대비 실적 분석
- 문서 및 소송 관리

## 업무 시작 순서

신규 프로젝트를 시작할 때는 일반적으로 다음 순서로 진행합니다.

### 1. 프로젝트 등록

- 프로젝트명
- 사업 유형
- 시행 기간
- 기본 정보 입력

### 2. 상품 구조 설정

- 계약 그룹
- 타입
- 동/호
- 옵션

### 3. 계약 조건 설정

- 공급가
- 계약금
- 중도금
- 잔금

### 4. 토지 매입 정보 등록

- 부지 목록
- 소유자
- 계약 정보

### 5. 계약 등록

- 고객 정보
- 계약 정보
- 유닛 매칭

### 6. 수납 및 회계 처리

- 고객 납부 등록
- 입출금 등록
- 전표 생성

### 7. 예산 및 보고서 확인

- 예산 대비 실적
- 계약 현황
- 자금 현황
