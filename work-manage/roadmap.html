로드맵 (Roadmap) ​

로드맵 페이지는 프로젝트의 주요 마일스톤(단계/버전)별 진행 상황을 시각적으로 확인하고 관리하는 화면입니다. 프로젝트를 여러 단계로 나누어 각 목표 대비 현재 어느 정도 도달했는지 파악할 수 있습니다.

1. 개요 ​

프로젝트의 전체 일정을 큰 단위(단계)로 나누어 관리합니다. 각 단계에는 여러 개의 **업무(Issue)**가 연결되어 있으며, 이 업무들의 완료 상태가 모여 해당 단계의 전체 진행률을 결정합니다.

2. 주요 기능 안내 ​

📊 단계별 진행도 확인 ​

  • 진행바(Progress Bar): 각 단계 카드에는 전체 업무 중 완료된 업무의 비율이 그래프로 표시되어 직관적인 확인이 가능합니다.
  • 상태 요약: 단계별로 '전체 업무 수', '완료된 업무', '진행 중인 업무' 건수를 숫자로 보여주어 상세 수치를 파악할 수 있습니다.

🔍 단계 상세 조회 및 업무 확인 ​

  • 특정 단계의 제목을 클릭하면 해당 단계의 상세 정보 화면으로 이동합니다.
  • 연결된 업무: 해당 단계에 할당된 모든 업무 목록을 확인할 수 있습니다. 완료된 업무는 취소선으로 표시되어 구분하기 쉽습니다.
  • 설명: 단계 생성 시 입력한 목표나 세부 설명을 확인할 수 있습니다.

➕ 단계 관리 (추가/수정/삭제) ​

  • 새 단계 추가: 우측 상단의 [새 단계] 링크를 클릭하여 프로젝트의 새로운 마일스톤을 정의합니다.
  • 편집 및 삭제: 단계 상세 화면 상단에서 기존 단계의 정보를 수정하거나, 필요 없는 단계를 삭제할 수 있습니다.

⚙️ 설정 연동 ​

  • 화면 우측 상단의 더보기(점 세 개 아이콘) 메뉴를 통해 프로젝트 설정의 [단계] 탭으로 즉시 이동하여 모든 마일스톤을 한꺼번에 관리할 수 있습니다.

3. 화면 구성 상세 ​

진행률 계산 방식 ​

  • 해당 단계에 연결된 모든 업무의 **완료 비율(Done Ratio)**을 합산하여 평균을 낸 수치입니다. 모든 업무가 100% 완료되면 단계의 진행률도 100%가 됩니다.

단계 상태(Status) ​

  • 진행 중 (Open): 현재 활발히 업무가 진행되고 있는 단계입니다.
  • 잠김 (Locked): 업무 추가는 제한되나 진행 상황은 계속 추적하는 단계입니다.
  • 닫힘 (Closed): 모든 목표가 달성되어 종료된 단계입니다.

4. 이용 팁 ​

  • 업무 할당: 새로운 업무를 만들거나 수정할 때 '목표 버전' 또는 '단계' 필드에서 특정 마일스톤을 선택하면 로드맵에 자동으로 반영됩니다.
  • 빠른 업무 관리: 상세 화면의 업무 목록 우측 드롭다운을 이용하면 페이지 이동 없이 업무 상태를 즉시 변경할 수 있어 편리합니다.

5. 로드맵 메뉴가 보이지 않나요? (노출 조건) ​

프로젝트 메뉴에서 '로드맵' 탭이 보이지 않는다면 다음 사항을 확인해 보세요.

  • 등록된 단계(버전) 없음: 프로젝트에 등록된 '단계'가 하나도 없을 경우 로드맵 메뉴는 자동으로 숨겨집니다. [설정 > 단계] 메뉴에서 새로운 단계를 하나 이상 생성하면 메뉴가 나타납니다.
