업무 프로젝트 ​

업무 관리 시스템의 메인 화면인 프로젝트 목록 페이지에 대한 사용자 가이드입니다. 이 페이지에서는 진행 중인 다양한 업무 프로젝트를 확인하고 관리할 수 있습니다.

1. 화면 개요 ​

프로젝트 목록 화면은 사용자가 참여 중이거나 열람 가능한 모든 프로젝트를 보드(카드) 또는 목록(테이블) 형태로 보여줍니다. 상단의 검색 도구와 보기 방식 전환 옵션을 통해 효율적인 프로젝트 관리가 가능합니다.

2. 주요 기능 안내 ​

🔍 프로젝트 검색 및 필터링 ​

  • 화면 상단의 검색 바를 이용해 프로젝트 이름으로 원하는 프로젝트를 빠르게 찾을 수 있습니다.
  • 특정 조건(상태, 공개 여부 등)을 선택하여 목록을 필터링할 수 있어 필요한 정보만 선별하여 조회할 수 있습니다.

➕ 새 프로젝트 생성 (관리자 전용) ​

  • 프로젝트 관리 권한(workManager)을 가진 사용자는 우측 상단의 [새 프로젝트] 링크를 통해 새로운 프로젝트를 생성할 수 있습니다.
  • 일반 사용자의 경우 해당 버튼이 노출되지 않으며, 할당된 프로젝트 내에서의 활동만 가능합니다.

🖼️ 보기 방식 전환 (보드 / 목록) ​

검색 조건 하단의 [옵션] 섹션에서 원하는 보기 방식을 선택할 수 있습니다.

  • 보드 방식: 각 프로젝트를 하나의 카드로 표현하여 핵심 정보를 시각적으로 보여줍니다. 화면 크기에 따라 레이아웃이 자동으로 최적화됩니다.
  • 목록 방식: 프로젝트 이름, 식별자, 설명을 정돈된 테이블 형태로 보여줍니다.
    • 계층 구조 시각화: 상위 프로젝트와 하위 프로젝트의 관계가 들여쓰기와 아이콘으로 명확하게 표시됩니다.
    • 상태 배지: 잠금보관된 프로젝트는 별도의 배지로 구분되어 쉽게 식별할 수 있습니다.

💡 설정 보존 기능 ​

  • 사용자가 선택한 보기 방식(보드 또는 목록)은 브라우저에 자동으로 저장됩니다.
  • 페이지를 새로고침하거나 나중에 다시 접속하더라도 이전에 사용했던 보기 방식을 그대로 유지하여 최적의 사용자 경험을 제공합니다.

🏷️ 아이콘 및 범례 안내 ​

화면 하단에서는 프로젝트의 상태를 나타내는 아이콘의 의미를 확인할 수 있습니다.

  • 내 프로젝트 (): 본인이 담당자로 지정되어 있거나 구성원으로 참여 중인 프로젝트를 나타냅니다.
  • 내 북마크 (): 사용자가 즐겨찾기에 추가하여 별도로 관리 중인 프로젝트를 나타냅니다.

3. 이용 팁 ​

  • 프로젝트의 전체적인 트리 구조를 파악하고 싶을 때는 '목록' 방식을, 요약 정보를 시각적으로 훑어보고 싶을 때는 '보드' 방식을 추천합니다.
  • 프로젝트 이름이나 카드를 클릭하면 해당 프로젝트의 상세 업무(이슈), 추진 현황(로드맵), 캘린더 등 세부 메뉴로 이동할 수 있습니다.
