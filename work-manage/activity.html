실행기록 (Activity) ​

실행기록 페이지는 프로젝트 내에서 발생한 모든 변경 사항과 활동 내역을 시간 순으로 확인할 수 있는 '프로젝트 히스토리' 화면입니다. 누가, 언제, 어떤 업무를 수행했는지 한눈에 파악할 수 있으며, 각 활동의 핵심 내용을 요약하여 제공합니다.

1. 화면 개요 ​

실행기록은 프로젝트의 살아있는 기록부와 같습니다. 업무의 생성 및 상태 변경, 의견 작성뿐만 아니라 회의록, 공지사항, 문서, 게시글의 업데이트 내역까지 날짜별로 정리되어 표시됩니다. 기본적으로 최근 10일간의 기록을 먼저 보여주며, 각 항목의 상세 페이지로 들어가지 않고도 주요 내용을 미리 확인할 수 있는 컨텐츠 요약 기능을 제공합니다.

2. 주요 기능 안내 ​

📅 기간별 기록 조회 및 이동 ​

  • 기간 확인: 목록 상단에 표시된 날짜 범위를 통해 현재 보고 있는 기록의 기간을 확인할 수 있습니다.
  • 날짜 이동: 목록 하단의 [« 뒤로] 버튼은 10일 이전의 기록으로, [다음 »] 버튼은 10일 이후의 기록으로 이동합니다.
  • 특정 날짜 지정: 우측 설정창의 '10일 기록' 달력 아이콘을 클릭하여 특정 날짜를 선택하면, 해당 날짜를 기준으로 과거 10일간의 기록을 즉시 불러옵니다.

👥 사용자별 활동 필터링 ​

  • 특정 팀원의 활동만 모아보고 싶을 때 우측 설정창의 [사용자] 필터를 사용합니다.
  • 본인의 활동 내역만 확인하려면 목록에서 **[<< 나 >>]**를 선택하고 '적용' 버튼을 클릭하세요.

🗂️ 활동 유형별 필터링 (다각화된 카테고리) ​

보고 싶은 활동의 종류만 선택하여 목록을 간소화할 수 있습니다. 각 유형별로 다음과 같은 정보를 제공합니다.

  • 업무: 업무 생성, 상태 변경 등의 활동과 함께 해당 업무의 설명 요약을 표시합니다.
  • 의견: 업무에 달린 댓글 내용을 직접 확인할 수 있습니다.
  • 회의록: 새 회의록 등록 시 회의 의제(Agenda) 요약을 함께 표시합니다.
  • 공지: 공지사항의 요약(Summary) 또는 내용을 확인할 수 있습니다.
  • 문서: 업로드되거나 수정된 문서의 본문 요약을 표시합니다.
  • 게시물: 게시판에 작성된 게시글의 본문 요약을 표시합니다.

🌲 하위 프로젝트 포함 ​

  • 현재 프로젝트에 포함된 하위 프로젝트들의 활동까지 함께 보고 싶다면 [하위 프로젝트] 체크박스를 선택하세요. 대규모 프로젝트의 전체 흐름을 파악할 때 유용합니다.

3. 이용 팁 ​

🔍 일관성 있는 컨텐츠 미리보기 ​

최근 업데이트를 통해 모든 활동 유형(업무, 의견, 회의, 공지, 문서, 게시물)에서 컨텐츠 요약이 일관성 있게 제공됩니다. 이태릭체로 표시되는 요약 문구를 통해 상세 페이지 이동 없이도 활동의 맥락을 빠르게 파악할 수 있습니다.

🔗 상세 내용 바로가기 ​

  • 활동 기록에 표시된 제목(업무 번호, 회의 제목, 문서명 등)을 클릭하면 해당 항목의 상세 페이지로 즉시 이동합니다.
  • 사용자 이름을 클릭하면 해당 사용자의 프로필 및 상세 정보 페이지로 이동합니다.

📈 히스토리 추적 및 투명성 ​

특정 업무의 상태가 변경되었거나 중요한 결정이 내려진 시점을 찾고 싶을 때 실행기록을 활용하세요. 모든 활동에는 작성자가 명시되므로 팀 내 커뮤니케이션의 투명성을 높여줍니다.
