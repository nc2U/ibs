납부 내역 관리 ​

프로젝트의 전체 대금 수납 현황을 조회하고, 입금된 내역을 각 계약 건에 매칭하여 관리하는 페이지입니다.

  • 위치: 계약 납부 관리 > 납부 내역 관리

개요 ​

회계 장부(원장)에 기록된 입금 내역을 바탕으로 타입별 수납 요약 정보를 확인하고, 각 입금 건별로 어떤 계약자의 몇 회차 납부금인지 연결(매칭)합니다. 계약 정보가 누락되거나 회차가 지정되지 않은 수납 건을 별도로 필터링하여 관리할 수 있습니다.

주요 기능 ​

  • 타입별 매출 예산 대비 분양 및 수납 현황 요약 제공
  • 입금 일자, 계좌, 차수/타입 등 다양한 조건으로 수납 내역 검색
  • 계약 미등록 또는 회차 미지정 수납 건 매칭 및 확인
  • 수납 건별 또는 계약자별 납부 현황 엑셀 다운로드
  • 매칭 완료된 수납 건 클릭 시 해당 계약자의 상세 납부 관리 페이지로 이동

화면 구성 ​

1. 납부 현황 요약 ​

현재 프로젝트의 매출 목표 대비 실제 분양 및 수납 성과를 타입별로 집계하여 보여줍니다.

주요 표시 항목 ​

  • 총 매출예산(A): [수입 예산 등록]에 설정된 전체 수입 목표액
  • 총 분양금액(B): 현재까지 체결된 모든 계약의 총 공급가 합계
  • 총 수납금액(C): 실제로 입금 처리된 금액의 합계
  • 미 수납금액(B-C): 계약은 체결되었으나 아직 입금되지 않은 잔액
  • 미 분양금액(A-B): 매출 예산 중 아직 계약되지 않은 잔여 목표액

2. 목록 제어 및 필터링 ​

원하는 수납 내역을 찾기 위해 다양한 검색 조건을 설정할 수 있습니다.

주요 입력 항목 ​

라벨필수입력 여부비고
납부일자 (From/To)특정 기간 내 입금된 내역 필터링
차수 / 타입특정 계약 그룹이나 평형별 입금 내역 조회
납부회차 선택계약금, 중도금 등 특정 회차로 입금된 내역 조회
납부계좌 선택특정 분양 수납 계좌로 입금된 내역 조회
계약 미등록 (Switch)장부에는 입금되었으나 아직 계약자와 연결되지 않은 내역만 조회
회차 미등록 (Switch)계약자와 연결은 되었으나 납부 회차가 지정되지 않은 내역만 조회
검색계약자, 입금자, 적요, 비고 내용으로 검색

3. 대금 납부 현황 목록 ​

필터링된 수납 내역을 리스트 형태로 표시하며, 각 건의 매칭 상태를 시각적으로 보여줍니다.

  • 노란색 배경: 계약 정보가 연결되지 않은 수납 건 (매칭 필요)
  • 빨간색 텍스트: 납입 회차가 확인되지 않은 수납 건 (확인 필요)

주요 기능 ​

  • 계약 건 매칭: 계약 정보가 없는 건의 [확인] 버튼을 클릭하여 수동으로 계약자를 찾아 연결할 수 있습니다.
  • 상세 이동: 매칭이 완료된 건의 계약자명이나 금액을 클릭하면 해당 계약자의 [건별 납부 관리] 페이지로 이동합니다.

4. 엑셀 다운로드 ​

조회된 데이터를 두 가지 형태의 엑셀 파일로 저장할 수 있습니다.

  • 수납건별: 리스트에 표시된 개별 입금 내역을 그대로 엑셀로 추출합니다.
  • 계약자별: 각 계약자별로 현재까지의 총 납부 현황을 집계하여 추출합니다. (입력된 종료일자 기준)
